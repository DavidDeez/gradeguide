# AI-Grader Pro — Full Project Roadmap (SDLC Phases)

**Project Name:** AI-Grader Pro  
**Methodology:** Waterfall SDLC (with iterative prototyping for UI)  
**Start Date:** May 2026

## Phase Overview

| # | Phase | Deliverables | Status |
|---|-------|-------------|--------|

---

## Phase 1: Requirements & Specification
- Software Requirements Specification (SRS) document
- Functional & non-functional requirements
- Project scope and objectives
- Stakeholder identification
- Approval/signature page

## Phase 2: System Design
- Use Case Diagram (UML)
- Data Flow Diagram (DFD) — Level 0 and Level 1
- Process Flow Diagram
- System Architecture Diagram
- Database schema design
- Technology stack selection rationale

## Phase 3: Implementation
- Project scaffolding and environment setup
- User authentication system (login/register)
- Course material upload module (PDF, DOCX)
- AI marking engine (RAG-based, grounded in uploaded content only)
- Dashboard and results display
- HCI-compliant responsive UI

## Phase 4: Testing
- Unit tests for backend modules
- Integration tests for AI pipeline
- UI/UX testing checklist
- Performance and load testing notes
- Bug tracking and resolution log

## Phase 5: Documentation
- User manual (with screenshots)
- Full code documentation (inline + API docs)
- Cost estimation breakdown
- Development standards followed

## Phase 6: Deployment & Presentation
- Deployment guide
- Final presentation slides (15–20 slides)
- Live demo preparation
- Project timeline summary chart
